# Darkdot API 

JavaScript API wrappers for Substrate and IPFS specific to Darkdot blockchain and DeCommerce

## Install

Using npm:

```bash
npm i @darkpay/dark-api
```

Using yarn:

```bash
yarn add @darkpay/dark-api
```
